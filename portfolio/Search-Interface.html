Search Interface

July 2014 — August 2014

I worked on the search functionality within an application that manages press releases. Based on my research, most search boxes display a dropdown list containing recommendations of what to search. This is great for users who are not exactly certain what they are looking for. However, users of this application usually know what press release they want to find. Therefore, I decided to implement live search. Live search shows actual results in the dropdown list instead of suggestions; saving users several clicks and making the search process faster.

For: A+E Television Networks

Role: Information Technology Intern

Type: Private Company Application

Tools: OmniGraffle
